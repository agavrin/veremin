Veremin

Видео-терменвокс, позволяющий сочинить замечательную музыку, просто размахивая руками!

Usage

После этой страницы вы увидите на экране информацию, извлеченную из видео - ключевые точки тела человека, обнаруженные с помощью нейронной сети PoseNet. Видео разделено на две части - двигайте руками в каждой из них, чтобы издавать звуки.

Перемещайте правую руку вверх и вниз (в правой зоне), чтобы изменять мелодию.

Перемещайте левую руку вверх и вниз (в левой зоне), чтобы изменять мелодию.

Requirements

At a minimum, your browsers must allow access to a webcam and support the Web Audio API.

If your browser supports the Web MIDI API, you can connect a MIDI synthesizer to your computer or you can download and run a software synthesizer such as SimpleSynth.

Depending on your browser, you may need to access this application using the https protocol instead of the http.

Check out the README for more information.
